BURNWOOD

Burnwood is an open source, cross-platform tool used to generate Windows Exploitation Packages (A Malware). The tool is capable of generating Windows Executables consisting of a malware masking behind any normal file. The results could be used to study the vulnerabilities, and ultimately will help you prevent them. Burnwood provides two delivery methods, Email Based (Gmail) and Cloud Based (Google's Firebase). Both of these methods works on Google Services and has it's own pros and cons. EXPLORE

KEYLOGGER BUILDER

Keylogger Builder is a standalone application to build multiple keyloggers. A Keylogger Builder is an application using which you could build an EMAIL based Keylogger specifically for Windows. It is FUD and capable of running in the background ever after every reboot. Please note that this Keylogger isn't a RAT and cannot be stopped remotely unless the user themselves removes it manually. You'll have to change your emails password in order to stop receiving the logs. EXPLORE

BYPASS DEFENDER

A script to bypass Windows Defender in order to carryout certain tasks. Bypass Defender is a BASH script used to bypass Windows Defender. You can bypass applications with specific extensions or a specific application by defining it's abolute path or the filename. It is extremely useful to carryout a few tasks which Windows Defender doesn't allow by default. EXPLORE
